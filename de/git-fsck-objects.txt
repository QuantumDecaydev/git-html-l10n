git-fsck-objects(1)
===================

NAME
----
git-fsck-objects - überprüft die Verbundenheit und Korrektheit der Objekte
in der Datenbank


SYNOPSIS
--------
[verse]
'git fsck-objects' ...

DESCRIPTION
-----------

Dies ist ein Synonym für linkgit:git-fsck[1]. Bitte lesen Sie die
Dokumentation dieses Kommandos.

GIT
---
Teil der linkgit:git[1] Suite
