git-init-db(1)
==============

NOM
---
git-init-db - Crée un dépôt Git vide


SYNOPSIS
--------
[verse]
'git init-db' [-q | --quiet] [--bare] [--template=<répertoire_de_modèles>] [--separate-git-dir <répertoire git>] [--shared[=<permissions>]]


DESCRIPTION
-----------

Ceci est un synonyme pour linkgit:git-init[1].  Veuillez vous référer à la
documentation de cette commande.

GIT
---
Fait partie de la suite linkgit:git[1]
